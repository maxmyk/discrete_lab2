# discrete_lab2



## Installation

```bash
git clone https://github.com/maxmyk/discrete_lab2
```

## Usage
Start `one` server and `multiple` clients
### Start server
```bash
cd discrete_lab2
python server.py
```
### Start client
open new terminal
```bash
cd discrete_lab2
python server.py
```
